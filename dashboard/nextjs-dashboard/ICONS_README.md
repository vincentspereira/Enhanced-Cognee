# Icon Generation

This directory needs the following icon files:

## Required Icons

1. **favicon.ico** - 16x16, 32x32
2. **favicon-16x16.png** - 16x16
3. **favicon-32x32.png** - 32x32
4. **apple-touch-icon.png** - 180x180
5. **icon-16.png** - 16x16 (for manifest)
6. **icon-32.png** - 32x32 (for manifest)
7. **icon-180.png** - 180x180 (for manifest)
8. **icon-192.png** - 192x192 (for manifest)
9. **icon-512.png** - 512x512 (for manifest)
10. **og-image.png** - 1200x630 (Open Graph image)
11. **screenshot-wide.png** - 1280x720 (PWA screenshot)
12. **screenshot-narrow.png** - 750x1334 (PWA screenshot)

## Quick Generation

Using a tool like [Favicon.io](https://favicon.io) or [RealFaviconGenerator](https://realfavicongenerator.net):

1. Create a 512x512 PNG logo
2. Upload to generator
3. Download all generated sizes
4. Place in this directory

## Design Guidelines

- Use the Enhanced Cognee logo
- Blue color scheme (#3b82f6)
- Clean, modern design
- White background for icons
- Transparent background where appropriate

## Icon Content

The Enhanced Cognee logo should feature:
- Brain or neural network icon
- Clean lines
- Modern aesthetic
- Scalable design
